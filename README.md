# ApexcifyTechnologys_Apexify-website
📌 Project Overview
This project is a responsive company website designed for ApexcifyTechnologys using HTML, CSS, and JavaScript.
The website includes multiple sections such as Home, About, Services, and Contact, and provides a clean and modern user interface.
This project was developed as part of my Web Development Internship at ApexcifyTechnologys.

🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
  
⚙ Features
- Responsive website design for all devices
- Navigation system using JavaScript (Next / Previous functionality)
- Lightbox view for images
- Smooth scrolling and transitions
- Hover effects for better user experience
- Clean and modern UI layout

🚀 How to Run the Project
1. Download or clone this repository
2. Open the project folder
3. Run **index.html** in any web browser
